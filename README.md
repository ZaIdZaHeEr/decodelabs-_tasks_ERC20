# ERC20 Token Smart Contract

A simple ERC20 token implementation built using Solidity and OpenZeppelin Contracts for learning and educational purposes.  
This project demonstrates how fungible tokens work on the Ethereum blockchain, including minting, transfers, approvals, and supply control.

---

# Features

- ERC20 standard token implementation
- Custom token name and symbol
- Fixed maximum supply (max supply cap)
- Owner-controlled minting
- Token transfers between users
- Allowance and approval system
- Balance tracking per wallet
- Secure OpenZeppelin-based implementation

---

# Technologies Used

- Solidity ^0.8.20
- OpenZeppelin Contracts
- Foundry (for development & testing)
- Ethereum Virtual Machine (EVM)
